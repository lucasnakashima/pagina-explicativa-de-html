Client x Server

Clients

    No desenvolvimento moderno de software, os Clients compreendem o hardware e o software que durante o seu funcionamento acessam recursos disponibilizados por um Server. Esse software pode ser tanto uma aplicação instalada em um computador quanto uma acessada pela Internet com o auxílio de um navegador. Exemplos de Client incluem...

No caso de aplicações instaladas:

  • Loja de aplicativos (Play Store)
  • Aplicativos de mensagem instantânea (Whatsapp, Facebook)
  • E-mail (Outlook, Gmail)

No caso de aplicações na Web

  • Plataformas de vídeo (Youtube)
  • Sites de banco
  • Redes sociais (Twitter, Facebook)

Nesse sentido, é importante mencionar que um mesmo software pode ser estar disponível tanto como aplicação instalada quanto como aplicação web

Servers

    Os Servers, nesse mesmo contexto, compreendem o hardware e o software que atendem as requisições por recursos realizadas pelos Clients. Como esse trabalho geralmente é pesado, o hardware deles costuma ser robusto e, assim como o software no qual é executado, voltado a esse tipo de tarefa. Assim, possuem sistemas operacionais e computadores próprios diferentes daqueles dos Clients.

Linguagens de Programação

    As Linguagens de Programação são línguas formais concebidas para a descrição das instruções de programas de computador, expressando o seu algoritmo. Elas são concebidas para atender necessidades e problemas que surgiram ao longo do desenvolvimento da tecnologia e de suas aplicações, podendo ser voltadas para determinada área. Dessa forma, existem línguas voltadas as aplicações do Client, enquanto outras são direcionadas ao Server
